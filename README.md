# 🛠️ Universal Nara Tools - All-in-One Assistant

**Universal Nara Tools** is a lightweight, powerful, and comprehensive userscript designed to enhance your browsing experience on any website. It combines visual enhancements, media controls, security tools, and productivity features into a single, elegant floating interface.

> *"Fidelis usque ad finem"*

### ✨ Key Features

#### 🎨 Visual & Interface
* **Smart Visual Web (Dark Mode):** Intelligently inverts colors to create a dark mode on any website, saving your eyes at night.
* **Media Tools (Image Filters):** Customize the web's look with real-time sliders for **Contrast, Brightness, Saturation, and Sepia**.
* **Clean View (Aggressive):** Remove sticky headers, floating ads, popups, and overlays that clutter your screen.
* **Fullscreen Mode:** Toggle full-screen immersion with a single click.

#### ⏱️ Productivity & Utility
* **🚀 Instant Countdown (NEW):** Bypasses annoying wait timers (e.g., "Wait X seconds to download") by aggressively speeding up the countdown clock.
* **Auto Scroll:** Automatically scroll down long pages/articles with adjustable speed.
* **Dashboard Info:** Displays real-time Clock, Date, and your current **IP Location**.
    * *Note:* Uses a smart cache system. Click the IP text to force a refresh (VPN-friendly).

#### 📺 Video & Media Controller
* **Universal Speed Control:** Speed up or slow down HTML5 videos/audio on any site (Unfortunately, it still doesn't work for Youtube) from **0.1x to 5.0x**.
* **Picture-in-Picture (PiP):** Force enable PiP mode for videos to watch while multitasking.

#### 🛡️ Security & Privacy
* **Copy/Paste Bypass:** Re-enable the right-click menu, text selection, and copy/paste on sites that block them.
* **Link Sanitizer:** Automatically removes tracking parameters (e.g., `utm_source`, `fbclid`, `gclid`) from URLs for cleaner sharing and better privacy.

#### 🌐 Translation
* **Integrated Translator:**
    * **Web Mode:** Translate the entire page using Google Translate.
    * **Mini Mode:** A quick popup to translate snippets of text between languages without leaving the page.

### ⚙️ How to Use
1.  Install the script.
2.  Look for the **Floating Button (Gear Icon)** in the bottom-right corner of any website.
3.  Click to open the **Nara Tools Menu**.
4.  Toggle features on/off. Settings are **saved automatically per website**.
5.  Drag the menu or button to reposition them anywhere on the screen.

### 🔧 Customization
* **Themes:** Choose from Yellow, Blue, Green, Red, Purple, or pick your own **Custom Color**.
* **Opacity:** Adjust the transparency of the menu.
* **Draggable UI:** The menu and button remember their position.
